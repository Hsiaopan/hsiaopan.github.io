MacOS通过终端SSH连接腾讯云服务器

2020-06-29

通过SSH连接云服务器

  • 创建SSH密匙对
    在云服务器左侧菜单栏，点击SSH密匙，可以创建密匙

  • 绑定实例，下载私钥文件

  • 创建 ~/.ssh 目录，用来存放密匙文件

    cd
mkdir .ssh

  • 将私钥文件移动到 ~/.ssh 文件夹内

  • 修改私钥文件权限，赋予私钥文件仅本人可读权限

    chmod 400 <私钥文件绝对路径>

  • 在 ~/.ssh 下创建 config 文件，内容如下，并保存

    Host Tencent               // 实例名称，可自定义
HostName 192.*.*.*          // 服务器IP
User root                   // 输入登录账号
Port 22                     // 输入端口号，默认22
IdentityFile ~/.ssh/Tencent //输入私钥文件绝对路径

  • 运行命令连接至实例

    ssh Tencent

如出现如下提示

“REMOTE HOST IDENTIFICATION HAS CHANGED! ”

出现这个问题的原因是公钥对不上，~/.ssh/know_hosts 文件里存在同IP的旧密钥，清理掉重新连接服务器即可解决。
